# SLF4J (API + Simple Provider)

## Decision

SLF4J is the standard logging facade for Java. `slf4j-simple` is used as the provider per project conventions. It requires no configuration file beyond `simplelogger.properties` and has no transitive dependencies beyond the API itself.
